Architecture

Core Engine (Bash)
  • scanner.sh - Main scanner engine
  • modules/ - Pluggable check modules
  • Native Linux tools integration
  • POSIX-compliant scripting
Web Interface (Python Flask)
  • app.py - Flask web application
  • templates/ - Jinja2 HTML templates
  • static/ - CSS, JS, assets
  • SQLite database backend
Why This Architecture?

The scanner uses Bash for core security checks because it provides direct access to Linux system internals and doesn't require external dependencies. The Flask web interface adds modern usability while maintaining the robust Bash backend for actual security validation.

Getting Started

Quick Start
  1. Clone the repository
  2. Make scripts executable: chmod +x scanner.sh
  3. Run basic scan: ./scanner.sh filesystem
  4. View results in terminal output
Web Interface
  1. Install Python dependencies: pip install -r requirements.txt
  2. Start web app: python app.py
  3. Open browser to http://localhost:5000
  4. Click "Start New Scan" to begin
